このディレクトリは、過去のPRドキュメントを整理するために使用されていましたが、現在は重要なエッセンスを speckit の適切な場所にまとめています。

## 方針

**重要PRのエッセンスを汎用的なガイドラインとして抽出**

具体的なPRドキュメントは削除し、そこから得られた重要な知見を以下のドキュメントにまとめています：

- [`.specify/memory/development-guidelines.md`](../../.specify/memory/development-guidelines.md): 開発ガイドラインとベストプラクティス
- [`.specify/scripts/pr-documentation-policy.md`](../../.specify/scripts/pr-documentation-policy.md): PRドキュメント保持方針
